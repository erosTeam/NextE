# NextE W01-W12 icon variants

Formal wide-spectrum color candidates for the selected candidate-05 shape.
Geometry is unchanged across all variants.

Current default:

- `W02 Porcelain Ink`
- `AppScope/resources/base/media/app_icon.png`
- `AppScope/resources/base/media/app_icon_sunset_solid_background.png`
- `AppScope/resources/base/media/app_icon_sunset_solid_foreground.png`
- `entry/src/main/resources/base/media/startup_foreground.png`

Packaged previews:

- `AppScope/resources/base/media/app_icon_nexte_pages_wide_variants_preview.png`
- `AppScope/resources/base/media/app_icon_nexte_pages_wide_variants_size_preview.png`

Optional layered resources are named:

- `AppScope/resources/base/media/app_icon_nexte_pages_w01_layered.json`
- `AppScope/resources/base/media/app_icon_nexte_pages_w02_layered.json`
- ...
- `AppScope/resources/base/media/app_icon_nexte_pages_w12_layered.json`

To switch the default later, copy the selected variant's flat, background, and
foreground PNGs to the compatibility resource names above.
